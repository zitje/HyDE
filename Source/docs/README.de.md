FÃ¼r die UnterstÃ¼tzung von Nixos gibt es ein eigenes Projekt, das unter [Hydenix] (<https://github.com/richen604/hydenix/tree/main>) gepflegt wird.

> [!IMPORTANT]

> Vergewissern Sie sich, dass Ihre NVIDIA-Karte die dkms Treiber aus der [hier] bereitgestellten Liste unterstÃ¼tzt (<https://wiki.archlinux.org/title/NVIDIA>).

> [!CAUTION]


> Sie kÃ¶nnen auch andere Anwendungen, die Sie zusammen mit HyDE installieren mÃ¶chten, in `Scripts/pkg_user.lst` einfÃ¼gen und die Datei als Parameter Ã¼bergeben, um sie wie folgt zu installieren:
